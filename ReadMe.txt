		==== Приложение "B.I.Calendar". Информация о версиях. ====
		==========================================================

 v. 0.1.1.xxx	(0.0 - 0.1)
============================

	+ Первая версия. Получила название "Календарь Мастер Pro".
	+ Разработана основная концепция дизайна, которая сохранилась и поныне.

 v. 0.1.2.xxx	(0.2 - 0.3)
============================

	# Усовершенствован дизайн (добавлены кнопки с названием дня, месяца и года, "пустые кнопки").
	# Наконец-то убран RESIZE. Теперь окно программы имеет строгие неизменяемые размеры.
	# Переработан движок календаря.
	# Сокращено название. Отныне программа называется "Календарь Pro".

	* Исправлены критические ошибки в основном движке.

	+ Программа наконец-то получила первую официальную иконку.

 v. 0.1.3.xxx	(0.4 - 0.5)
============================

	# Небольшие усовершенствования в движке календаря.
	# Доработан внешний вид. Добавлены кнопки с номерами недель.
	# Отныне выходные дни и номера недель выделены серым цветом.

	* Изменена иконка. Она получила свой нынешний облик.

	+ Наконец-то реализована иконка в трее! Она получила меню, позволяющее управлять приложением.
	+ Приложение получило MUTEX. Теперь нельзя запустить более одной копии приложения.
	+ Добавлен движок расчёта дня недели.

 v. 0.1.4.xxx	(0.6 - 0.7)
============================

	# Усовершенствовано меню. Появился пункт "Языки".
	# Доработан дизайн меню.
	# Исходный код программы занял 400 с лишним строк... А именно - 459 строк.

	* Исправлен движок календаря.
	* Полностью переделан движок меню.

	+ Наконец-то добавлена поддержка языков!!! Теперь можно подключать новые языки к программе,
		добавив всего один .lng файл в папку Languages. Первые языки, вошедшие в программу:
		Русский (Ru), Английский (En), Украинский (Ua) и Немецкий (De).

 v. 0.1.5.xxx	(0.8 - 0.9)
============================

	# Исходный код программы вырос до 931-и строк.
	# Все стандартные диалоги заменены на диалоги "собственного изготовления".
	# Теперь язык можно переключить "на лету". Либо поменять язык после завершения текущего
		сеанса работы программы.
	# .Lng файлы отныне поддерживаются в кодировке "Unicode", что позволило сохранять строки,
		содержащие символы национальных алфавитов.

	* Исправлена критическая ошибка в движке смены языков, приводившая к полному краху программы.
	* Исправлена критическая ошибка в чтении "главного языка" из файла. По умолчанию принят
		Английский (English (EN)).
	* Исправлена небольшая ошибка контроля повторного запуска.
	* Исправлен движок языковой поддержки. Теперь приложение не падает при отсутствии языковых
		файлов либо файлов настроек.

	+ Добавлен диалог "About...", отображающий текущую версию и дату создания программы.
	+ Добавлены новые языки - Молдавский (Md), Белорусский (BY) и Китайский (CN).
	+ Добавлены иконки к пунктам меню программы. Теперь интерфейс стал красивее.
	+ Наконец-то удалось избавиться от папки "Images". Теперь графические ресурсы хранятся
		прямо в .exe файле.

 v. 0.1.6.xxx	(1.0)
============================

	# Улучшен дизайн. Теперь внешний вид стал намного интуитивнее и понятнее.
	# Введена поддержка тем Windows XP, Vista и 7.
	# Теперь программа может похвастаться полной языковой поддержкой.
	# "Назойливый" .tmp файл стал появляется только во время работы программы и удаляется
		сразу при завершении.
	# Отныне размеры окна программы стали постоянными и не зависят даже от параметров,
		переданных при запуске.

	* Исправлена ошибка, приводившая к сбросу языковых настроек.

	+ Левая верхняя кнопка теперь работает так же, как и правый клик по иконке в трее -
		вызывает меню программы.
